## Особенности

| Механика           | Описание                                                |
| ------------------ | ------------------------------------------------------- |
| **Боевая система** | Гибрид реального времени с тактической замедлением      |
| **Отряд**          | Управляй группой уникальных персонажей целых 2 на выбор |
| **Тактика**        | Используй окружение, фланги и синергию способностей     |
| **RPG-элементы**   | Прокачка, лут, крафт и выбор пути развития              |
| **Ретро-стиль**    | Ностальгическая эстетика с душой                        |
